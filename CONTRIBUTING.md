## Project setup

- Node.js >= 20
- Install dependencies: `npm install`
- Run tests: `npm test`

---

## Tests

- Run all tests: `npm run test`
- Test status is reflected by the CI workflow and CI badge
- Check code coverage: `npm run coverage`
- Coverage is collected via Vitest and reported to Codecov
- Coverage status is reflected by the Codecov badge
- Please add or update tests for any behavior change

---

## Code style

- TypeScript strict mode is enabled
- ESLint checks must pass
- Code comments must be written in en-US

---

## Pull requests

- Fork the repository and create a dedicated branch for each change
- One logical change per pull request
- Include tests when changing behavior
- Keep pull requests small and focused
